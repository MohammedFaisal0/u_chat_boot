# 🤖 Uni Chat Bot - نظام المحادثة الذكي للجامعة

نظام محادثة ذكي متطور لطلاب جامعة الإمام محمد بن سعود الإسلامية، يوفر مساعدة أكاديمية وتقنية شاملة باستخدام الذكاء الاصطناعي.

## 🌟 المميزات الرئيسية

### 👨‍💼 لوحة تحكم الإدارة

- **إدارة المستخدمين:** إضافة وإدارة الطلاب وأعضاء هيئة التدريس
- **إدارة التعليمات:** إضافة وتعديل تعليمات البوت الذكي
- **إدارة المواد التعليمية:** مراجعة وموافقة المواد المقدمة من أعضاء هيئة التدريس
- **إحصائيات شاملة:** متابعة نشاط الطلاب والمحادثات والمواد

### 👨‍🏫 لوحة تحكم أعضاء هيئة التدريس

- **تقديم المواد:** رفع المواد التعليمية (PDF) للمراجعة
- **متابعة المواد:** عرض حالة المواد المقدمة
- **لوحة تحكم تفاعلية:** إحصائيات ومعلومات شخصية

### 👨‍🎓 لوحة تحكم الطلاب

- **محادثة ذكية:** تفاعل مع البوت الذكي للحصول على المساعدة
- **إدارة المحادثات:** إنشاء وحفظ المحادثات المهمة
- **واجهة متعددة اللغات:** دعم العربية والإنجليزية

### 🤖 البوت الذكي

- **ذكاء اصطناعي متقدم:** استخدام Groq API للردود الذكية
- **تعليمات قابلة للتخصيص:** الإدمن يمكنه إضافة تعليمات مخصصة
- **استجابة فورية:** ردود سريعة ودقيقة على استفسارات الطلاب

## 🛠️ التقنيات المستخدمة

- **Frontend:** Next.js 14, React, Tailwind CSS
- **Backend:** Next.js API Routes, Node.js
- **قاعدة البيانات:** MongoDB, Mongoose
- **الذكاء الاصطناعي:** Groq API (Llama 3.1)
- **المصادقة:** JWT (JSON Web Tokens)
- **الترجمة:** نظام i18n مخصص
- **معالجة الملفات:** PDF text extraction

## 📋 متطلبات النظام

- **Node.js:** الإصدار 18 أو أحدث
- **MongoDB:** الإصدار 4.4 أو أحدث
- **npm:** الإصدار 8 أو أحدث
- **Groq API Key:** للحصول على الذكاء الاصطناعي

## 🚀 التثبيت والتشغيل

### 1. تحضير البيئة

# تثبيت المتطلبات

npm install

````

### 2. إعداد متغيرات البيئة

أنشئ ملف `.env.local` في المجلد الرئيسي:

```env
# قاعدة البيانات
MONGODB_URI=mongodb://localhost:27017/uni-chat-bot

# مفتاح الذكاء الاصطناعي (Groq)
GROQ_API_KEY=your-groq-api-key-here

# مفتاح التشفير
JWT_SECRET=your-super-secret-jwt-key-here
````

### 3. الحصول على Groq API Key

1. اذهب إلى [Groq Console](https://console.groq.com/)
2. سجل حساب جديد أو سجل دخول
3. اذهب إلى "API Keys" في القائمة الجانبية
4. اضغط "Create API Key"
5. انسخ المفتاح وضعه في `.env.local`

### 4. تشغيل قاعدة البيانات

#### خيار 1: MongoDB محلي (Windows)

```bash
# تشغيل MongoDB على البورت الافتراضي
mongod

# أو تشغيل على بورت مخصص
mongod --port 27018 --dbpath C:\data\db

# إنشاء مجلد البيانات إذا لم يكن موجوداً
mkdir C:\data\db
```

#### خيار 1.1: MongoDB محلي (Linux/Mac)

```bash
# تشغيل MongoDB
sudo systemctl start mongod

# أو تشغيل يدوياً
mongod --dbpath /var/lib/mongodb
```

#### خيار 1.2: MongoDB مع Docker

```bash
# تشغيل MongoDB باستخدام Docker
docker run -d -p 27017:27017 --name mongodb mongo:latest

# أو مع مجلد بيانات مخصص
docker run -d -p 27017:27017 -v mongodb_data:/data/db --name mongodb mongo:latest
```

#### خيار 2: MongoDB Atlas (سحابي) - **الأفضل للنشر**

1. **أنشئ حساب على [MongoDB Atlas](https://www.mongodb.com/atlas)**
2. **أنشئ cluster جديد (مجاني)**
3. **احصل على connection string**
4. **ضعه في `.env.local`:**

```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/uni-chat-bot
```

**مميزات MongoDB Atlas:**

- ✅ **مجاني** حتى 512MB
- ✅ **لا يحتاج تثبيت** MongoDB محلياً
- ✅ **يعمل على أي جهاز** بدون إعداد
- ✅ **نسخ احتياطي تلقائي**
- ✅ **أمان عالي**

## 🐛 استكشاف الأخطاء

### مشاكل شائعة وحلولها

#### البوت لا يرد

- تأكد من صحة `GROQ_API_KEY` في `.env.local`
- تحقق من اتصال الإنترنت
- راجع logs في terminal

#### خطأ في قاعدة البيانات

- تأكد من تشغيل MongoDB
- تحقق من `MONGODB_URI` في `.env.local`
- أعد تشغيل `node setup-instructions.js` و `node setup-admin.js`

#### مشاكل في المصادقة

- تأكد من وجود `JWT_SECRET` في `.env.local`
- امسح cookies في المتصفح
- أعد تسجيل الدخول
